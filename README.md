### 2.版本说明

- v0.1.0
  - 定时器触发单个ADC0双通道采样，PA27和PA26，但是无法使用DMA
- v0.2.0
  - 单个定时器触发ADC0和ADC1同时采样，精度较高，但是会因为中断产生不连续点
- v0.3.0
  - 完成自适应采样，能够准确采集5个周期的波形，但不知道为什么会有不连续点
  - 下一步加入串口屏
- v0.4.0
  - 用串口屏取代了按键和LCD，增加了波形显示功能
  - 测得10kHz~10Hz能完美采出5个周期的波形，过大采的周期会变多，过小采的周期会变小
  - 5Hz~50kHz频率检测非常准确
  - 下一步测试FFT分析
- v0.5.0
  - 尝试加入滤波
  - 均值滤波效果不错，覆盖原数组与存为新数组对方波以外的波形影响不大。必要时可以使用二级均值滤波，效果更好
  - 中值滤波对方波效果很好，但对其他波形效果不如均值滤波。进行两次中值滤波和只进行一次区别不大
- v0.6.0
  - 测试发现双通道同步采样就是会产生一个不连续点，但是切换成单通道就没问题了
  - 彻底删除ADC1中断相关代码后不连续点消失
