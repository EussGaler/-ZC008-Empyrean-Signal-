# [ZC008]2025华大九天杯D题

### 1.功能描述

1. 实现双路ADC同时采样

### 2.版本说明

- v0.1.0
  - 定时器触发单个ADC0双通道采样，PA27和PA26，但是无法使用DMA

- v0.2.0
  - 单个定时器触发ADC0和ADC1同时采样，精度较高，但是会因为中断产生不连续点
- v0.3.0
  - 完成自适应采样，能够准确采集5个周期的波形，但不知道为什么会有不连续点
  - 下一步加入串口屏
- v0.4.0
  - 用串口屏取代了按键和LCD，增加了波形显示功能
  - 测得10kHz~10Hz能完美采出5个周期的波形，过大采的周期会变多，过小采的周期会变小
  - 下一步测试FFT分析
